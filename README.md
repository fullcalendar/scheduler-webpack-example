# FullCalendar Scheduler Webpack Example

An example dev environment for FullCalendar Scheduler and [Webpack].


## Installation

```sh
npm install
npm run build
```

Then run `index.html` in a browser.

For watching, run `npm run watch`.


[Webpack]: https://webpack.js.org/
